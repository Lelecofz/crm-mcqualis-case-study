# assets/

Capturas de ecrã usadas no README do case study. Para ficarem com bom aspeto:

- Formato **PNG**, largura ~1200–1600px.
- Tirar com **dados fictícios** (nunca dados reais de clientes).
- Nomes esperados pelo README:
  - `dashboard.png` — Dashboard com indicadores
  - `kanban.png` — Funil de processos (Kanban)
  - `cliente.png` — Ficha de cliente + linha do tempo
  - `portal.png` — Portal do cliente (vista do processo)

Depois de as adicionar aqui, troca os blocos "_Capturas de ecrã (a adicionar)_"
do `README.md`/`README.en.md` por imagens, por exemplo:

```markdown
![Dashboard](assets/dashboard.png)
```
